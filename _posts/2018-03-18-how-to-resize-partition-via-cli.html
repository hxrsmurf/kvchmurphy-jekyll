This is a follow-up to this same tutorial using Google Compute Engine's Tutorial (here). You will want to take a snapshot or ensure you have good backups in case you muck this up! What we're doing is deleting the partition, re-creating it at the same start sector, and then using the last end sector. Then checking the file system and finally resizing the partition.

I found this method was a lot quicker than going to GPARTED for it.

  1. fdisk /dev/sdb
  2. p
  3. d
  4. n
  5. [ENTER A FEW TIMES FOR DEFAULT]
  6. w
  7. q
  8. e2fsck /dev/sdb1
  9. e2fsck -f /dev/sdb1
  10. resize2fs /dev/sdb1
  11. mount /dev/sdb1 /mnt/seafile
