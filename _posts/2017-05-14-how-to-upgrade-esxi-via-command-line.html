  1. Navigate to VMWare's Patch Portal
  2. Download the latest update
  3. Upload to the ESXi Datastore
  4. SSH into the ESXi host and run the below

esxcli software vib update -d /vmfs/volumes/datastore1/ESXi650-201704001.zip
