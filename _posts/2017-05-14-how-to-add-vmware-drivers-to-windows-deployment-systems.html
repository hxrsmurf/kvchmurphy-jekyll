In Windows Deployment Systems (WDS):

  1. Expand Servers
  2. Expand the WDS Server
  3. Right-click Drivers
  4. Select Add Driver Package
  5. Select individual INF files or the whole folder (I do the whole folder)
  6. Browse and Navigate to one of the VMWare folders
  7. Next
  8. OK

The x64 and x86 folder locations, respectively:

  • C:\Program Files\Common Files\VMware
  • C:\Program Files (x86)\Common Files\VMware
