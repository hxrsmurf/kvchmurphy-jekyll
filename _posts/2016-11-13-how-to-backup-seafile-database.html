The official guide is great. Below is a breakdown of their steps for my environment (ESXi 5.5).

  1. Install Linux Mint Debian
  2. Install Updates
  3. Install VMWare Driver
  4. Install mysql-server, phpmyadmin, python-mysqldb, and screen
  5. Login to phmyadmin and create user seafile with special password
  6. Create 3 databases that are used by seafile: ccnet-db, seafile-db, seahub-db
  7. Grant seafile user to these 3 databases
  8. Import the SQL backups
  9. Start Seafile
